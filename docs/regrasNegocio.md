# Regras de Negócio
RN01. Cada usuário deve possuir um e-mail único para cadastro no sistema.  
RN02. O aluno deve realizar a Avaliação de Nível antes de iniciar sua primeira trilha.  
RN03. O nível do aluno deve ser determinado com base na pontuação obtida na Avaliação de Nível.  
RN04. Uma trilha de aprendizagem deve estar associada a uma única área de aprendizado.  
RN05. Um exercício objetivo deve possuir pelo menos uma alternativa correta.  
RN06. O progresso (%) do aluno só pode ser incrementado mediante conclusão válida de aula ou exercício.  
RN07. O streak só é incrementado quando houver atividade registrada no dia corrente.  
RN08. O XP é acumulativo e não pode ser reduzido por ações comuns do usuário.
